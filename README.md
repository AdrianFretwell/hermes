1. Install [Vagrant](https://www.vagrantup.com)
1. Install [Virtualbox](https://www.virtualbox.org)
1. Clone repository
   ```
   git clone git@gitlab.com:cecid/hermes.git
   ```
1. Start vagrant box
   ```
   cd hermes
   vagrant up
   ```
1. Point browser to:
   ```
   http://localhost:18080
   ```
